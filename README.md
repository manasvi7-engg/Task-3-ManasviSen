AI Recommendation System

Overview

This project is a simple AI-powered Movie Recommendation System developed as part of the DecodeLabs Artificial Intelligence Internship Program.

The system recommends movies to users based on their interests using Content-Based Filtering. It analyzes movie genres, converts textual data into numerical vectors using CountVectorizer, and calculates similarity scores using Cosine Similarity to find the most relevant recommendations.

Features

- Movie recommendation based on user preferences
- Content-based filtering approach
- Cosine similarity for recommendation logic
- Case-insensitive user input handling
- Beginner-friendly implementation
- Command-line interface

Technologies Used

- Python
- Pandas
- Scikit-Learn
- CountVectorizer
- Cosine Similarity

How It Works

1. User enters a movie name.
2. The system converts movie genres into numerical vectors.
3. Cosine similarity is calculated between movies.
4. The most similar movies are identified.
5. Top recommendations are displayed to the user.
